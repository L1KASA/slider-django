# Космическое агентство (Landing Page)

Лендинговая страница космического агентства, выполненная по макету из Figma.

## О проекте

Данный проект представляет собой одностраничный сайт с информацией о космическом агентстве. Включает в себя:
- Адаптивную верстку по макету Figma.
- Интерактивный слайдер фотографий с миниатюрами.
- Просмотр изображений в полном размере через Fancybox.
- Секции с преимуществами и информацией.

## Стек технологий

- **Django 6.0**: Основной Backend-фреймворк.
- **Bootstrap 5**: Отзывчивая верстка и компоненты.
- **jQuery**: Библиотека для работы слайдера.
- **Slick Slider**: Реализация основного слайдера и ленты миниатюр.
- **Fancybox 5**: Галерея и просмотр изображений.
- **Pillow & Easy-thumbnails**: Обработка и оптимизация изображений.

## Как запустить проект

1. **Клонируйте репозиторий:**
   ```bash
   git clone <url_репозитория>
   cd django-cosmo
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/macOS:
   source venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Выполните миграции:**
   ```bash
   python manage.py migrate
   ```

5. **Создайте суперпользователя (для доступа в админку):**
   ```bash
   python manage.py createsuperuser
   ```

6. **Запустите сервер:**
   ```bash
   python manage.py runserver
   ```

После этого сайт будет доступен по адресу `http://127.0.0.1:8000`. Админка доступна по адресу `http://127.0.0.1:8000/admin`.
